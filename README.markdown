# Installation

## Auto Install
**Please install HomeBrew and Zim first!**
```sh
bash <(curl -s https://raw.githubusercontent.com/hungdinhvan/dotfiles/master/install.sh)
```

# Update
Simply running a `git pull` inside the cloned folder

```sh
$ cd ~/.dotfiles
$ git pull
```
